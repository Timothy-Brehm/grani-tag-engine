# @grani/schema-editor

JSON Schema–backed file editor (Vite + React + MUI), moved into the grani-tag-engine workspace.

## Scripts

```bash
npm run dev -w @grani/schema-editor
npm run build -w @grani/schema-editor
npm run lint -w @grani/schema-editor
npm run preview -w @grani/schema-editor
```

From the workspace root: `npm run dev:schema-editor`.

Uses `@grani/schema-tools` for `Schema` typing and validation messages.
